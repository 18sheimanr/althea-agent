Privacy Policy – Personal AI SMS Assistant

Effective Date: February 28, 2026

This Privacy Policy describes how the Personal AI SMS Assistant (“Service”) collects, uses, and protects information. This Service is operated by Service Operator as a private, personal-use system and is not offered to the public.

1. Authorized User

This Service is restricted to a single authorized phone number:

+1 555 555 0100

Messages from any other number are rejected and not processed.

2. Information Collected

The Service collects and stores:

  • SMS message content
  • Sender phone number
  • Message timestamps
  • Event data created by the user (full day, partial day, reminder)
  • Basic message metadata required for processing

3. How Information Is Used

Information is used solely to:

  • Generate AI responses
  • Log and manage personal events
  • Send reminders
  • Provide search-based assistance
  • Maintain system functionality

The Service does not use data for advertising, profiling, analytics resale, or marketing purposes.

4. Data Sharing

Mobile information will not be shared, sold, rented, or disclosed to third parties for marketing or promotional purposes.

No user data is shared with external organizations except as required to operate the Service infrastructure (e.g., cloud hosting and SMS delivery providers).

5. Data Storage & Security

Data is stored securely using Google Cloud services, including Firestore. Reasonable technical safeguards are implemented to prevent unauthorized access.
